# Security Policy

## Supported Release Scope

- Current GA target: macOS builds.

## Reporting a Vulnerability

- Open a private security report through repository security advisories.
- Include reproduction steps, impact, and affected version.

## Vulnerability SLA (Default)

- Critical: 24 hours
- High: 72 hours
- Medium: 14 days
- Low: next planned release

## Release Blocking Policy

- Known unresolved Critical/High vulnerabilities block release unless explicitly waived.
